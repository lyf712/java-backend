## 二、动态规划

### 基本概念
>动态规划的目标是寻求最优解问题（最优解或其值），其中解决过程中存在着重复的子问题，
> 需要去通过递归公式在递归子过程的过程保存了计算过的子问题。

### 基本步骤

1. 刻画最优解的结构特征（关键步骤，从小规模递推）
2. 递归地定义**最优解的值**
3. 计算最优解的值，一般采用**自底向上**
4. 利用计算的信息构造最优解
